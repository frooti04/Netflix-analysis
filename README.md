# Netflix-analysis
Netflix analysis using Excel
## Objective
To analyze Netflix dataset and understand content distribution and trends.
## Dataset
Show ID, Title, Type, Country, Release year, Genre, Rating
## Tools Used
Microsoft Excel
## Analysis Performes
-Movies and TV Shows comparison
-Genre popularity analysis
-Country wise content distribution
-Rating analysis using Pivot Tables

Built interactive dashboard with charts and filters
